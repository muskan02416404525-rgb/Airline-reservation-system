# Airline-reservation-system
The Airline Reservation System is designed to simplify airline ticket booking and management. It enables users to search for available flights, reserve seats, manage passenger details, and generate booking records efficiently.

The project demonstrates the use of Java Swing for the graphical user interface and MySQL for secure and efficient database management.

🚀 Features
User Login and Authentication
Flight Management
Passenger Registration
Flight Search
Ticket Booking
Ticket Cancellation
Booking History
Seat Availability Check
Database Connectivity using JDBC
User-Friendly Graphical Interface


🛠️ Technologies Used
Java
Java Swing
JDBC
MySQL
NetBeans IDE
🗄️ Database


Database: MySQL
